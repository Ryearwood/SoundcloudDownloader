How to Use

1

Fork the Repository

Click the Fork button on GitHub to create your own copy of the repository.

2

Run the Workflow

Go to Actions tab, select "Download Playlist", and click "Run workflow".

3

Enter Playlist URL

Paste your SoundCloud playlist URL (include the ?si= parameter from the share link).

4

Download Your Music

Once complete, download the artifact containing your MP3 files.
